📋 Lista de Tarefas

Este é um projeto de uma Lista de Tarefas (To-Do List) simples e interativa, desenvolvido com HTML, CSS e JavaScript.

🚀 Funcionalidades

✅ Adicionar tarefas à lista;

🔄 Marcar tarefas como concluídas;

❌ Remover tarefas da lista;

🎨 Interface amigável.
